What is an Organisation Profile Document (OPD)?

The OPD contains the organisation's full name, homepage, logo, dataset location and contact information for the equipment dataset. To make your data discoverable using OPDs you can create one as it is a simple process and is a RDF file which describes your organisation and open access documents.
